How It Works

From sensor to alert in milliseconds

01
🔬

Sensors Detect

DHT22, MQ-2, and IR flame sensors continuously monitor temperature, smoke concentration, and open flame radiation.

→
02
⚡

ESP32 Processes

The ESP32 microcontroller reads sensor data, applies thresholds, and triggers the buzzer alarm locally for immediate response.

→
03
📡

Data Transmitted

Sensor readings are sent over WiFi to this dashboard via HTTP REST API, updating in real-time every few seconds.

→
04
🚨

Alert Issued

When thresholds are exceeded, the system logs the event, updates the dashboard, and sends email/SMS notifications instantly.
