How It Works

From sensor to extinguishment in milliseconds

01
🔥

Flame Detected

IR flame sensor detects infrared radiation from open fire. Digital output goes LOW instantly.

→
02
⚡

ESP32 Reacts

ESP32 reads the pin and drives the relay HIGH — no cloud round-trip. Reaction is under 50ms.

→
03
💧

Pump Activates

Relay closes, water pump runs. Continues for 10s after flame gone to fully extinguish.

→
04
🌐

Dashboard Updates

ESP32 sends status to Vercel API proxy every 2s. Dashboard updates live from anywhere in the world.
